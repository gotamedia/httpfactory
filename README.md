A set of factory classes used for creation of **atoms/http** objects. This is an implementation of the [PSR-17](https://www.php-fig.org/psr/psr-17/) standard.

## Installation

Use [Composer](https://getcomposer.org/) to download **atoms/httpfactory**:
```bash
composer require atoms/httpfactory
```

## About

- **atoms/httpfactory** requires with PHP 7.0
